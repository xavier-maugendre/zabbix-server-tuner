# ZabbixTune

Simulateur de configuration `zabbix_server.conf` — génère une config 
optimisée selon ton infrastructure (hosts, items, proxies, BDD).

## Utilisation
Ouvrir directement dans le navigateur ou via https://zabbixtune.vercel.app

## Fonctionnalités
- Profils Lab / Small / Medium / Large / XLarge
- Support proxy actif/passif
- MySQL, PostgreSQL, TimescaleDB
- Export `.conf` prêt à l'emploi
- Compatible Zabbix 7.0 / 7.4
